CPU Reference

  1. 核心
  2. 线程(超线程技术(Hyper-Threading): 核心数*2=线程数)
  3. 频率(内核时钟频率，可理解为运算速度)
    1. 主频：平常CPU核心工作的频率
    2. 睿频(Turbo)：核心全力工作的频率，实质是多核切换成单核模式然后自动超频，既提高了日常使用的速度，又节能。
    3. 外频(系统总线频率FSB)：CPU跟外部(系统总线)接触沟通的频率，由主板提供
    4. 倍频：主频=外频*倍频
    5. 超频：通过手动提高外频或倍频来提高主频
  4. 架构：大致分为X86，IA64，RISC等
  5. 缓存：越大越好
  6. 制作工艺：nm
  7. 接口：通常代表针脚数量
  8. 功耗：
  9. 指令集：

AMD vs Intel

Intel I5-9600KF

  1. I5: CPU系列，数字越大性能越好
  2. 9: 第9代CPU
  3. 600: 这一代中的型号，数字越大性能越好
  4. K: 不锁倍频
  5. F: 不带核显

AMD R5-3600X

  1. R5: CPU系列，数字越大性能越好
  2. 3: 3代锐龙
  3. 600: 型号
  4. X: 官方超频
  5. 后缀不带G：不带核显

显卡

命名

[N卡] RTX 2060 SUPER 08G

  1. RTX
    1. GS: 高清标准显卡
    2. GT: GS加强版
    3. GTX: GS超级版
    4. RTX: 多了光线追踪技术的GTX
  2. 20: 数字越大越新
  3. 60: 数字越大越好
  4. SUPER
    1. SUPER: 超级版
    2. TI(>SUPER): 加强版
  5. 0: 0比A略强
  6. 08G: 显存大小

[A卡] RX 580 XT

  1. RX
    1. RX
    2. R
    3. HD
  2. 5: 系列
  3. 80: 数字越大越好
  4. XT: 超级版
